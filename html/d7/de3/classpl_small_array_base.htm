Detailed Description

template<typename T, plUInt16 Size>
class plSmallArrayBase< T, Size >

Implementation of a dynamically growing array with in-place storage and small memory overhead.

Best-case performance for the PushBack operation is in O(1) if the plHybridArray does not need to be expanded. In the worst case, PushBack is in O(n). Look-up is guaranteed to always be in O(1).
