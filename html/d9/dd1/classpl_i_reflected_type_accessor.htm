Detailed Description

Provides access to the properties of an plRTTI compatible data storage.

inline

Constructor for the plIReflectedTypeAccessor.

It is a valid implementation to pass an invalid handle. Note that in this case there is no way to determine what is actually stored inside. However, it can be useful to use e.g. the plReflectedTypeDirectAccessor to set properties on the engine runtime side without having the plPhantomRttiManager initialized.
