Detailed Description

Derive from this class if you want to define an attribute that replaces the property type widget.

Using this attribute affects both member properties as well as elements in a container but not the container widget. When creating a property widget, the property grid will look for an attribute of this type and use its type to look for a factory creator in plRttiMappedObjectFactory<plQtPropertyWidget>. E.g. plRttiMappedObjectFactory<plQtPropertyWidget>::RegisterCreator(plGetStaticRTTI<plFileBrowserAttribute>(), FileBrowserCreator); will replace the property widget for all properties that use plFileBrowserAttribute.
