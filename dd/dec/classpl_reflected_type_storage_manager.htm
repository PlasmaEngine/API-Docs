Detailed Description

Manages all plReflectedTypeStorageAccessor instances.

This class takes care of patching all plReflectedTypeStorageAccessor instances when their plRTTI is modified. It also provides the mapping from property name to the data storage index of the corresponding plVariant in the plReflectedTypeStorageAccessor.
