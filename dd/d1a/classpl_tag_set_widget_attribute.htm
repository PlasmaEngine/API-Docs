Detailed Description

Add this attribute to a tag set member property to make it use the tag set editor and define the categories it will use as a ; separated list of category names.

Usage: PL_SET_MEMBER_PROPERTY("Tags", m_Tags)->AddAttributes(new plTagSetWidgetAttribute("Category1;Category2")),
