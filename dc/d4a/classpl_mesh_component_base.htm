Detailed Description

Base class for components that render static or animated meshes.

◆ DeserializeComponent()

void plMeshComponentBase::DeserializeComponent ( plWorldReader & inout_stream)
overridevirtual

Override this to load the current state of the component from the given stream.

The active state will be automatically serialized. The 'initialized' state is not serialized, all components will be initialized after creation, even if they were already in an initialized state when they were serialized.

Reimplemented from plComponent.

Reimplemented in plAnimatedMeshComponent, plInstancedMeshComponent, and plProcVertexColorComponent.

◆ SetSortingDepthOffset()

void plMeshComponentBase::SetSortingDepthOffset ( float fOffset)

The sorting depth offset allows to tweak the order in which this mesh is rendered relative to other meshes.

This is mainly useful for transparent objects to render them before or after other meshes.
