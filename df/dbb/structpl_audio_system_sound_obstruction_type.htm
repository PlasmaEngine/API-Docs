Detailed Description

The obstruction type applied to a sound. This affects the way that ray casting works for an audio source.

Member Enumeration Documentation

◆ Enum

enum plAudioSystemSoundObstructionType::Enum : StorageType
Enumerator
None 

No ray casting is done for this sound. The sound is neither obstructed nor occluded.

SingleRay 

Only one ray is shot at each frame. The occlusion value will be averaged over time. The sound will not be obstructed, since only one ray is not enough to compute this information.

MultipleRay 

Multiple rays are shot at each frame. The occlusion and obstructions values will be averaged over time.
